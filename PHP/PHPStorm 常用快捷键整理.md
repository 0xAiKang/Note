### 常用
```
智能代码补全: ⌃ ⇧ Space
在任何地方搜索：Double ⇧
全局搜索：⌘ ⇧ F
局部文件搜索：⌘ F
显示最近打开的文件记录：⌘ E
```

### 通用
```
打开系统偏好设置：⌘ ,
```

### 调试
```
进入下一步：F8/F7
智能进入下一步/跳出：⇧ F7/⇧ F8
运行到光标处：⌥ F9
弹出计算表达式：⌥ F8
恢复程序运行：⌘ ⌥ R
切换端点：⌘ F8
展示端点信息：⌘ ⇧ F8
```

### 查询/替换
```
查询任何东西：Double ⇧
文件内查找：⌘ F
全局查找：⌘ ⇧ F
全局替换：⌘ ⇧ R
选择所有匹配项：⌃ ⌘ G
```

### 编辑
```
基本代码补全：⌃ Space
智能代码补全：⌃ ⇧ Space
行末自动添加分号：⌘ ⇧ ↵

显示函数/变量简要信息：⌘ mouse

注释/取消行注释：⌘ /
注释/取消块注释：⌘ ⌥ /

连续选中代码块：⌥ ↑
减少当前选中的代码块：⌥ ↓
格式化代码：⌘ ⌥ L
自动缩进：⌃ ⌥ I

剪切当前行：⌘ X
打开剪切板：⌘ ⇧ V
复制当前行自动向下粘贴：⌘ D
删除当前行：⌘ ⌫
删除到单词的开头：⌥ ⌫
删除到单词的末尾：⌥ fn ⌫

智能将代码拼接成一行：⌃ ⇧ J

开始新的一行：⇧ ↵
折叠/展开代码块：⌘ - / ⌘

将单行代码向上/向下移动：⌘ ⇧ ↑ / ⌘ ⇧ ↓
```

### 重构
```
显示当前类的层次结构：⌃ H
显示当前方法的层次结构：⌘ ⇧ H
显示当前调用的层次结构：⌃ ⌥ H
```

### 导航
```
左右切换Tab 页面：⌃ ← / ⌃ →
导航到指定行：⌘ L
显示最近打开的文件记录列表：⌘ E
回到前一个/后一个光标所在位置：⌥ ⌘ ← / ⌥ ⌘ →
跳转到最后一次编辑的地方：⌘ ⇧ ⌫
进入光标所在的方法：⌘ click
跳转到接口实现类：⌘ ⌥ B
快速打开光标所在方法/类的定义：⌘ Y
跳转到下一个/上一个高亮错误位置：F2 / F12

回到上一个操作点：⌘ [
回到下一个操作点：⌘ ]
```

### 版本控制
```
提交代码：⌘ K
更新代码：⌘ T
```
